## 알고리즘 주요 유형

1. **자료구조**
   - 배열, 연결 리스트
   - 스택, 큐
   - 해시
   - 힙, 트리, 그래프

2. **기본 알고리즘**
   - 정렬 (퀵 정렬, 병합 정렬, 삽입 정렬 등)
   - 탐색 (이진 탐색, BFS, DFS)
   - 재귀

3. **고급 알고리즘**
   - 다이나믹 프로그래밍
   - 그리디 알고리즘
   - 백트래킹
   - 분할 정복

## 추천 학습 방법

1. 기본 자료구조와 알고리즘 개념 이해하기
2. 유형별 기본 문제 풀이
3. 실전 문제 풀이로 응용력 키우기
4. 시간 복잡도와 공간 복잡도 분석 연습
5. 정기적인 복습과 난이도 점진적 상승
