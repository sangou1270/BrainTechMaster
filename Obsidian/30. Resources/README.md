## Resources 관리 방법

1. Resources는 주제별로 폴더를 구성합니다.
2. 각 주제 폴더에는 README.md 파일이 있어야 하며 해당 주제의 개요를 포함해야 합니다.
3. 주제별 리소스들은 태그를 활용하여 분류하고 연결할 수 있습니다.
4. 더 이상 관심이 없는 Resources는 Archive로 이동할 수 있습니다.
